1. 用户管理模块

用户注册、登录、信息管理等接口

POST
/api/v1/users/register
用户注册

请求参数

{
  "username": "testuser",
  "password": "password123",
  "email": "test@example.com",
  "phone": "13800138000",
  "nickname": "测试用户",
  "avatar": "https://example.com/avatar.jpg"
}

响应示例

{
  "code": 200,
  "message": "success",
  "data": {
    "id": 123,
    "username": "testuser",
    "email": "test@example.com",
    "phone": "13800138000",
    "nickname": "测试用户",
    "avatar": "https://example.com/avatar.jpg",
    "status": "ACTIVE",
    "createTime": "2024-01-01T00:00:00",
    "updateTime": "2024-01-01T00:00:00"
  }
}
POST
/api/v1/users/login
用户登录

请求参数

{
  "username": "testuser",
  "password": "password123",
  "loginType": "USERNAME"
}

2. 标签管理模块

标签查询、管理等接口

GET
/api/v1/tags/list
标签列表查询

请求参数

参数名 类型 必填 描述
tagType String 否 标签类型（CONTENT、USER、CATEGORY）
status String 否 标签状态（ACTIVE、INACTIVE、DELETED）
keyword String 否 关键词搜索
currentPage Integer 否 当前页码（默认：1）
pageSize Integer 否 页面大小（默认：20）

9. 社交动态模块

发布动态、分享功能等接口

POST
/api/v1/social/dynamics/create
创建动态

请求参数

{
  "userId": 123,
  "content": "动态内容",
  "dynamicType": "TEXT",
  "mediaUrls": ["https://example.com/image1.jpg"],
  "location": "位置信息",
  "isPublic": true
}
POST
/api/v1/social/dynamics/share
分享功能

请求参数

{
  "userId": 123,
  "shareTargetId": 456,
  "shareTargetType": "CONTENT",
  "shareContent": "分享说明",
  "sharePlatform": "INTERNAL"
}

接口规范说明

统一响应格式：所有API都使用统一的响应格式，包含code、message、data字段

分页参数：currentPage从1开始，pageSize建议不超过100

认证方式：Bearer Token，请求头：Authorization: Bearer <token>

时间格式：ISO 8601标准格式 YYYY-MM-DDTHH:mm:ss
